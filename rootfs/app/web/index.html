企业微信API代理服务

代理服务已成功启动！您现在可以通过此服务访问企业微信API。

支持的API路径

本代理服务支持所有企业微信API，主要包括以下分类：

通用路径

  • 所有API： /cgi-bin/

身份验证

  • /cgi-bin/gettoken

成员管理

  • /cgi-bin/user/list
  • /cgi-bin/user/get
  • /cgi-bin/user/create
  • /cgi-bin/user/update
  • /cgi-bin/user/delete
  • /cgi-bin/user/batchdelete
  • /cgi-bin/user/simplelist
  • /cgi-bin/user/convert_to_openid
  • /cgi-bin/user/convert_to_userid
  • /cgi-bin/user/authsucc
  • /cgi-bin/user/getuserinfo

部门管理

  • /cgi-bin/department/list
  • /cgi-bin/department/get
  • /cgi-bin/department/create
  • /cgi-bin/department/update
  • /cgi-bin/department/delete

标签管理

  • /cgi-bin/tag/list
  • /cgi-bin/tag/create
  • /cgi-bin/tag/update
  • /cgi-bin/tag/delete
  • /cgi-bin/tag/get
  • /cgi-bin/tag/addtagusers
  • /cgi-bin/tag/deltagusers

消息推送

  • /cgi-bin/message/send

菜单管理

  • /cgi-bin/menu/create

素材管理

  • /cgi-bin/media/upload
  • /cgi-bin/media/get
  • /cgi-bin/media/uploadimg

应用管理

  • /cgi-bin/agent/list
  • /cgi-bin/agent/get
  • /cgi-bin/agent/set

外部联系人

  • /cgi-bin/externalcontact/
  • /cgi-bin/externalcontact/get_follow_user_list
  • /cgi-bin/externalcontact/add_contact_way
  • /cgi-bin/externalcontact/get_contact_way
  • /cgi-bin/externalcontact/update_contact_way
  • /cgi-bin/externalcontact/del_contact_way

通讯录管理

  • /cgi-bin/batch/invite
  • /cgi-bin/batch/syncuser
  • /cgi-bin/batch/replaceuser
  • /cgi-bin/batch/replaceparty
  • /cgi-bin/batch/getresult

会话内容存档

  • /cgi-bin/msgaudit/

企业支付

  • /mmpaymkttransfers/
  • /cgi-bin/mch/

群机器人

  • /cgi-bin/webhook/send

使用方法

将您应用中的企业微信API请求地址从 https://qyapi.weixin.qq.com 修改为当前代理服务器地址即可。

注意：本代理仅转发请求，不存储任何数据。请确保您的应用正确处理API的鉴权和安全。
